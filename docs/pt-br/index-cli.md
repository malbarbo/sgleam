Sgleam é uma versão do [Gleam](https://gleam.run/) voltada para o ensino de programação. Ele oferece um REPL interativo, suporte a imagens e programas interativos.

Este guia cobre o uso do sgleam pela linha de comando. Para o uso pelo navegador, imagens e programas interativos, veja o [Guia Web](web.pt-br.html).
